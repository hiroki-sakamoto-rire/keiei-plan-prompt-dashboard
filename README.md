# 経営計画書プロンプト作成ダッシュボード

金融機関に提出する経営計画書のたたき台を作るための、社外共有可能な静的ダッシュボードです。

## 公開URL

GitHub Pagesなどで公開したURLをここに記載します。

```text
https://hiroki-sakamoto-rire.github.io/keiei-plan-prompt-dashboard/
```

## 使い方

1. 公開URLをブラウザで開きます。
2. 顧問先の基本情報、資金使途、直近数値、3年後の目標を入力します。
3. 右側の「生成プロンプト」をコピーします。
4. ChatGPTなどに貼り付けて、経営計画書ドラフトを作成します。
5. 出てきた内容を、決算書・試算表・資金繰り表・返済予定表と照合して修正します。

## 社外共有について

このダッシュボードは静的ファイルだけで動くため、GitHub Pages、Netlify、Vercelなどで公開できます。入力内容はブラウザ内だけで処理され、サーバーには保存されません。

公開に必要なファイルは以下です。

- `index.html`
- `styles.css`
- `script.js`

## 入力時のコツ

- 売上は「年商」だけでなく、可能なら客数・単価・件数に分解します。
- 利益改善策は「何を、いつまでに、誰がやるか」まで書きます。
- 金融機関には、希望額だけでなく「返済原資」と「資金使途の妥当性」が伝わるようにします。
- 数値が曖昧な場合は、無理に埋めずに「確認が必要」として扱います。

## 次に本格化する場合

- Excel入力から自動で数値計画を作る
- 3か年損益計画、資金繰り表、返済計画を出力する
- 業種別のテンプレートを追加する
- 金融機関面談用の説明メモを自動生成する
- WordまたはPDF形式の提出用計画書に変換する

## 注意

このツールは経営計画書の作成補助です。金融機関への提出前には、実際の決算書、試算表、借入明細、資金繰り表と整合しているかを必ず確認してください。
